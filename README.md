# Sons of Mukla - Assignments

[![GitHub Pages](https://img.shields.io/badge/pages-live-0b8f46)](https://allingaming.github.io/somassigns/)
[![Flutter Web](https://img.shields.io/badge/flutter-web-02569b)](https://flutter.dev)

A lightweight Flutter web app that renders raid assignments pulled live from the shared Google Sheet.

## Highlights
- **Boss selector:** Gnarlmoon, Anomalus, Chess, Sanv, Rupturan, Kruul, Mephistroth.
- **Live data:** Reads directly from the sheet without rebuilds.
- **Search:** Full-table highlighting for matches (Anomalus and others), Gnarlmoon side context.
- **Visuals:** Boss profile art, Rupturan phase images with tap-to-zoom.
- **Notes:** Boss-specific Kara40 callouts in a prominent style.
